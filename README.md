- **People**: Age, Gender, Income...
- **Health Metrics**: Cholesterol, BMI, Blood Pressure, CK-MB, Troponin, Blood Sugar...
- **Lifestyle Indicators**: Smoking, Alcohol, Exercise, Diet, Sedentary Time, Sleep...
